## Running the Application

1. Open the project with a IDE (ex:VS Code)
2. Navigate to src -> envs -> env.ts
3. Comment line 4 (with EC2 link)
4. Uncomment line 3 (with local URL)
5. Install necessary packages by

```bash
npm i
```

6. Start the app

```bash
ng serve
```
